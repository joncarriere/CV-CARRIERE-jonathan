La Roue Tourne
Projet JavaScript

La Roue Tourne

Projet de groupe à l'issue de notre module de formation JavaScript

Cahier des charges :

  • Site e-commerce comportant une page web unique
  • Menu permettant de naviguer entre les différentes catégories de produits
  • Affichage uniquement des produits de la catégorie sélectionnée
  • Bouton d'ajout au panier sous chaque fiche produit
  • Possibilité de modifier les quantités dans le panier, supprimer un produit
  • Affichage du prix total dans le panier

Languages et frameworks : HTML, CSS, JavaScript, Bootstrap

Ce projet fut très intéressant pour mettre en pratique nos connaissances acquises dans les différents languages étudiés jusque là. La difficulté majeure du projet a résidé dans la création du panier et malheureusement celui-ci est resté à un stade peu fonctionnel.

Le Meeple Savant

Le Meeple Savant

Projet personnel présenté pour le passage du Titre Professionnel

Le Meeple Savant est une centralisation d'explications de règles de jeux de société glanées sur les forums spécialisés. L'utilisateur trouvera facilement des solutions à des situations de jeu complexes rencontrées lors de parties de jeux de société. L'utilisateur pourra créer un compte afin d'ajouter du contenu au site : ajout d'un jeu à la base de données, ajouts de points de règles...
